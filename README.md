# Perimeter

A clock in/clock out app for healthcare workers, gated by a geofenced perimeter, with separate manager and care-worker experiences. Built as a take-home task for Lief's WebDev Engineer role.

> This README is being written progressively as the app is built. A full architecture overview, setup instructions, and a feature checklist (attempted vs. skipped) will be filled in before submission.

## Stack

Next.js (App Router, TypeScript) · GraphQL (Yoga + Apollo Client) · Prisma + PostgreSQL (Neon) · Auth0 · Ant Design · React Context · Chart.js · Vercel

## Local development

```bash
npm install
npm run dev
```
